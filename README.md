# Master's Thesis --- Pascal Rodrigo Anema (2026)
This repository contains the source code used for the experiments of the Master's Thesis "Memory Efficient B*", which is also provided in this repository.
The source code, in TeX, has also been provided for the Master's Thesis document.

The thesis was supervised by Prof. Dr. Mark H.M. Winands and Dr. Dennis J.N.J Soemers, and submitted on March 31st 2026.

Three branches are available of this repository to provide additional code. The "java17" branch contains the source code of this project modified to run on Java 17, as the default code base assumes Java 24 is available. The "extra" branch contains additional code for a new B*² class "BstarSquaredKeep", which implements all four bounds preservation techniques described in the thesis. Additionally, the "extra" branch implements the shallow and deep irrelevance stop-conditions, as described in the thesis. All experiments and their results, as described in the thesis, were performed using the "main" branch.
